# 현대자동차 비대면 시승 웹 어플리케이션

> hyundai mobile non-face-to-face test drive services.

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
